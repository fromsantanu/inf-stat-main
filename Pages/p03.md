## âœ¨ **2. Z-Test**

### ğŸ”¹ **2.1 One-sample Z-test**

âœ… **Use when:**

* Population variance (ÏƒÂ²) is known
* Sample size is large (n > 30)

âœ”ï¸ **Example:** Testing if the mean weight of apples is equal to 150g.

ğŸ”§ **Formula:**
$Z = \frac{\bar{x} - \mu}{\sigma / \sqrt{n}}$

ğŸ“ **Python Example:**

```python
import numpy as np
from scipy import stats

# Data
sample_mean = 152
pop_mean = 150
sigma = 5
n = 40

# Z-test calculation
z = (sample_mean - pop_mean) / (sigma / np.sqrt(n))
p_value = 2 * (1 - stats.norm.cdf(abs(z)))

print("Z:", z)
print("p-value:", p_value)
```

---

### ğŸ”¹ **2.2 Two-sample Z-test**

âœ… **Use when:**

* Comparing means of **two independent samples**
* Both population variances are known and samples are large

âœ”ï¸ **Formula:**
$Z = \frac{\bar{x}_1 - \bar{x}_2}{\sqrt{ \frac{\sigma_1^2}{n_1} + \frac{\sigma_2^2}{n_2} }}$

ğŸ”§ **Example:** Comparing average marks of two classes.

## âœ¨ **3. T-Test**

### ğŸ”¹ **3.1 One-sample t-test**

âœ… **Use when:**

* Population variance is **unknown**
* Sample size is small (n < 30)

âœ”ï¸ **Example:** Testing if average test score = 75.

ğŸ”§ **Python Example:**

```python

pop_mean = 75

t_stat, p_value = stats.ttest_1samp(sample, pop_mean)

print("t-statistic:", t_stat)
print("p-value:", p_value)
```

---

### ğŸ”¹ **3.2 Independent two-sample t-test**

âœ… **Use when:**

* Comparing means of two **independent groups**
* Population variances are unknown but assumed equal

âœ”ï¸ **Example:** Comparing test scores of males vs females.

ğŸ”§ **Python Example:**

```python

group2 = [80, 82, 84, 79, 77, 83]

t_stat, p_value = stats.ttest_ind(group1, group2)

print("t-statistic:", t_stat)
print("p-value:", p_value)
```

---

### ğŸ”¹ **3.3 Paired t-test**

âœ… **Use when:**

* Comparing **before-after** results on same subjects

âœ”ï¸ **Example:** Testing weight before and after diet program.

ğŸ”§ **Python Example:**

```python

after = [68, 66, 73, 78, 70]

t_stat, p_value = stats.ttest_rel(before, after)

print("t-statistic:", t_stat)
print("p-value:", p_value)
```

## âœ¨ **4. ANOVA (Analysis of Variance)**

âœ… **Definition:**
Used to test **differences between means of 3 or more groups**.

---

### ğŸ”¹ **4.1 One-way ANOVA**

âœ… **Use when:**

* Comparing means across **one factor with multiple groups**

âœ”ï¸ **Example:** Comparing exam scores among 3 teaching methods.

ğŸ”§ **Python Example:**

```python

group2 = [78, 80, 79]
group3 = [92, 94, 91]

print("F-statistic:", f_stat)
print("p-value:", p_value)
```

---

### ğŸ”¹ **4.2 Two-way ANOVA**

âœ… **Use when:**

* Testing effect of **two independent variables (factors)** on a dependent variable.

âœ”ï¸ **Example:** Effect of **teaching method** and **gender** on exam scores.

ğŸ”§ **Implementation:**
Use **statsmodels** for two-way ANOVA.

```python
import pandas as pd
import statsmodels.api as sm
from statsmodels.formula.api import ols

# Sample data
data = pd.DataFrame({
    'score': [88, 92, 85, 90, 78, 80, 79, 77, 92, 94, 91, 93],
    'method': ['A','A','A','A','B','B','B','B','C','C','C','C'],
    'gender': ['M','F','M','F','M','F','M','F','M','F','M','F']
})

model = ols('score ~ C(method) + C(gender) + C(method):C(gender)', data=data).fit()
anova_table = sm.stats.anova_lm(model, typ=2)
print(anova_table)
```

---

### ğŸ”¹ **4.3 Repeated Measures ANOVA**

âœ… **Use when:**

* Same subjects measured under **multiple conditions or times**.

âœ”ï¸ **Example:** Testing blood pressure at 3 time points (morning, afternoon, evening) in same patients.

ğŸ”§ **Implementation:**
Use **statsmodels** or **pingouin** for repeated measures ANOVA.

```python

```
